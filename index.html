Labs

Lab 1 - UI Design

Reflection: In this lab, I learned the foundational principles of user interface design using Jetpack Compose in Android development. I explored how composable functions are used to build UI elements such as text, buttons, images, and layouts. At first, I struggled to understand how different UI components interact within a single screen, but through practice, I gained clarity on structuring layouts effectively. This lab helped me understand the importance of simplicity, alignment, and spacing in creating user-friendly interfaces that enhance usability and visual appeal.

Lab 2 - Interaction

Reflection: In this lab, I learned how to implement user interaction features such as button clicks, text input handling, and event responses in Android applications. I understood how apps can respond dynamically to user actions, which is essential for creating engaging mobile experiences. Initially, I faced difficulty handling input validation and state updates, but after repeated practice, I became more comfortable with event-driven programming. This lab strengthened my understanding of how user interaction plays a key role in improving application usability and responsiveness.

Lab 3 - Material Design

Reflection: This lab introduced me to Material Design principles and how they are applied in Android applications to create consistent and visually appealing user interfaces. I learned about color schemes, typography, spacing, elevation, and component consistency. Through this lab, I realized that design is not only about appearance but also about improving user experience. Applying Material Design made my applications look more professional and easier to navigate, which is important in real-world mobile development.

Lab 4 - Navigation

Reflection: In this lab, I learned how to implement navigation between multiple screens in an Android application using Jetpack Compose navigation components. I also learned how to pass data between screens, which is important for building multi-page applications. At first, managing multiple screens was confusing, especially when handling data flow between them, but this lab helped me understand navigation architecture more clearly. It improved my ability to structure applications in a more organized and user-friendly way.

Lab 5 - Room Database

Reflection: In this lab, I learned how to use Room Database to store and manage local data in Android applications. I understood the concepts of entities, DAO (Data Access Objects), and database classes, which are essential for persistent data storage. Initially, I found database concepts quite challenging, but through practice, I gained a better understanding of how data is stored, retrieved, and updated within an application. This lab gave me valuable insight into real-world application development where data persistence is essential.
